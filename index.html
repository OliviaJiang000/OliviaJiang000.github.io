一、拿到一份 EDF 数据可以做哪些分析？
EDF（European Data Format）是用来存储生物电信号（如 EEG/ECG/EMG）的标准格式。当你拿到一份 EDF 文件，你可以按以下层次进行分析：

✅ 1. 数据结构探索（元信息 + 通道信息）
文件中包含哪些通道？（如 EEG Fpz-Cz、ECG、EMG）

每个通道的采样率是多少？

记录时长？注释有多少段？

包含标注（annotations）吗？如睡眠阶段、任务标签

📦 工具：raw.info, raw.ch_names, raw.annotations

✅ 2. 原始波形可视化与浏览
多通道波形图（查看干扰、异常、放电、闭眼等）

滤波前后比较（如带通 1–40Hz，去除漂移）

📦 工具：raw.plot(), raw.filter()

✅ 3. 频域分析
功率谱密度（PSD）：查看不同频段（δ、θ、α、β、γ）的能量

频带能量分布统计

📦 工具：raw.plot_psd(), psd_welch, fft, wavelet

✅ 4. 时间-频率分析（频谱热图）
展示信号在不同时间段频率分布变化

用于任务变化、睡眠阶段、癫痫发作检测等

📦 工具：tfr_multitaper, STFT, CWT

✅ 5. 通道比较/空间分析
多通道相关性分析（同步性、相干性）

Topomap 头皮地形图：显示某一频段能量在头皮的分布

📦 工具：mne.viz.plot_topomap(), spectral_connectivity()

✅ 6. 注释分析 / 事件相关分析（ERPs, Sleep stage）
标注信息的提取与分段（如 N1、N2、REM 等）

不同阶段的统计对比（如不同睡眠阶段的 PSD）

事件相关电位（ERP）

📦 工具：mne.read_annotations, raw.annotations, Epochs

✅ 7. 去伪迹处理（如眨眼、肌电）
使用 ICA 去除眼电、肌电等干扰

📦 工具：mne.preprocessing.ICA()

✅ 8. 特征提取 + 机器学习
提取：

均值、峰值、方差

频带功率、熵、相位同步

构建：

分类模型（如睡眠阶段、情绪识别、癫痫检测）

特征选择与通道压缩

📦 工具：sklearn, xgboost, cnn/lstm, tsfresh, mne_features

✅ 9. 可视化仪表盘构建
使用 Dash / Streamlit 构建交互式页面

支持通道选择、波形查看、频谱切换、事件对比等

📦 工具：plotly, dash, streamlit, holoviews

📦 二、EDF 原始数据的结构（Technical）
一个 EDF 文件本质上包含：

✅ 1. Header（元信息）
PatientID, RecordingID

记录时间、通道数、采样率

每个通道的物理单位（μV）、量化范围

✅ 2. Signal Data（电信号矩阵）
通道数 × 总采样点数

通常按通道叠加方式编码

✅ 结论：EDF 是什么？你能做什么？
EDF 文件 ≈ 多通道生物电信号 + 元信息 + 注释（可选）
你可以从信号质量控制 → 时频分析 → 空间分析 → 特征提取 → 可视化展示进行完整工作流，最终可以对接 AI 模型。
